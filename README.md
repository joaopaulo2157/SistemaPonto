# 📱 Ponto Eletrônico - Sistema de Registro de Ponto

## Sobre o Projeto

Sistema completo de ponto eletrônico com reconhecimento facial, desenvolvido como Progressive Web App (PWA) para instalação em celulares iOS e Android.

## Funcionalidades

- ✅ Reconhecimento facial com câmera
- ✅ Registro de ponto (entrada/saída)
- ✅ Gerenciamento de colaboradores
- ✅ Troca de turnos
- ✅ Área T.I com login
- ✅ Folha de ponto mensal para impressão
- ✅ Backup dos dados
- ✅ Funciona offline
- ✅ Instalação como app nativo

## Tecnologias

- HTML5 / CSS3 / JavaScript
- PWA (Progressive Web App)
- Vercel API (backend)
- Vercel Hosting
